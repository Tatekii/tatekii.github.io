脚手架搭建&Lerna新手包

本文最后更新于：2022年1月10日 晚上

脚手架工具

脚手架结构

  • 文件头部

    #!/usr/bin/env node

  • 命令结构

    vue <command>  [options]  --options params
<!-- // eg: -->
vue create my_app -p env=...

本地开发

  • 链接本地脚手架

      `npm link`全局node_modules中添加目前库

  • 链接本地脚手架的依赖库

    // lib dir 
npm link
// 同时注意在脚手架的package.json中手动 
dependencies:{
    "xxx-lib":"^1.0.0"
}
// cli dir
npm link xx-lib

  • 解开本地lib和本地包的链接

    // lib dir
npm unlink
// cli dir
rm -rf node_modules
npm unlink xxx-lib

命令

  • npm link/unlink链接当前项目到全局
  • npm link/unlink xxx-lib将xxx-li添加/从当前node_modules中移除

Lerna

基于Git+Npm的多package项目管理工具

  1. 解决原生脚手架开发中的重复操作

    • 多package本地link
    • 多package依赖安装
    • 多package单元测试
    • 多package代码提交
    • 多package代码发布

  2. 解决版本一致性

    • package版本一致
    • 相互依赖版本升级

lerna开发脚手架流程

  1. 项目创建

    npm init 
npm i lerna
lerna init

  2. 创建package

    lerna create // 创建package
lerna add //安装依赖
lerna //链接依赖

  3. 开发测试

    lerna exec // 执行shell脚本
lerna run
lerna clean // 清空依赖
lerna bootstrap // 重装依赖

  4. 发布上线

    lerna version // 提升版本号
lerna change //  查看变更
lerna diff // 查看diff
lerna public // 发布
