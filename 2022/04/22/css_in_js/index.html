Css In Js

优点

  • css有自己的作用域
  • 让组件更具可移植性，开箱即用
  • 让样式具有动态功能(参数)
  • 一个地方修改标签类型和标签参数
  • 相比style能写更多（伪类，级联）

缺点

  • 复杂度⬆️
  • class名混乱不语意
